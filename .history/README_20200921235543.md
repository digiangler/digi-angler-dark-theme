## Installation

1. Open the extensions sidebar on Visual Studio Code
1. Search for Digi-Angler Theme
1. Click Install
1. Click Reload to reload your editor
1. Select the Manage Cog (bottom left) > Color Theme ＞ Digi-Angler Theme
1. 🌟🌟🌟🌟🌟 Rate five-stars 😃
